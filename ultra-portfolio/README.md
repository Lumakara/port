### 3. Customisasi

#### Ganti Konten
Edit file `index.html` untuk mengganti:
- Nama dan informasi pribadi
- Foto profile
- Pengalaman kerja
- Skill dan teknologi
- Proyek portfolio
- Informasi kontak

#### Ganti Warna
Edit bagian `tailwind.config` di `index.html`:
```javascript
colors: {
    'primary': '#030412',
    'aqua': '#33c2cc',
    'fuchsia': '#ca2f8c',
    // ... warna lainnya
}
```
